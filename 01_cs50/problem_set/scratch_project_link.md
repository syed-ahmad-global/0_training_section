## Overview

This project was built using [Scratch](https://scratch.mit.edu), a visual block-based programming language developed by MIT. It was submitted as **Problem Set 0** of CS50 at Harvard Extension School the very first assignment of the course, designed to introduce core programming concepts through hands-on creation.

---

## CS50 Requirements Met

The project satisfies all of CS50's mandatory criteria for this problem set:

- **Two or more sprites** — at least one of which is not the default Scratch cat
- **Three or more scripts** — distributed across sprites to handle different behaviors
- **At least one conditional** — an `if/else` block that makes decisions based on some condition
- **At least one loop** — a `repeat` or `forever` block to control repeated actions
- **At least one variable** — used to track state, score, or some changing value
- **At least one custom block** — created via _Make a Block_, taking at least one input as an argument
